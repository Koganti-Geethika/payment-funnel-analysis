
# Payment Funnel Analysis Project

## Overview
This project analyzes user drop-offs across a payment funnel using Python, Flask, Pandas, and Streamlit.

## Funnel Stages
Visit → Add to Cart → Checkout → Payment Success

## Tech Stack
- Backend: Flask + Pandas
- Frontend: Streamlit + Plotly
- Data: CSV
